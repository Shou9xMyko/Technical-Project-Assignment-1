CRUD (Create Read Update Delete)

CRUD adalah singkatan dari Create, Read, Update, dan Delete. Ini adalah operasi dasar yang sering digunakan dalam pengembangan perangkat lunak dan aplikasi database. CRUD digunakan untuk mengelola data dalam sebuah sistem atau aplikasi

Dalam aplikasi yang lebih kompleks, ada beberapa operasi tambahan seperti search, filter, atau sort yang juga digunakan untuk mengelola data. Namun, CRUD merupakan dasar dari semua operasi data dalam aplikasi.

Aplikasi Web Fetch API

Fetch API adalah API baru yang diperkenalkan dalam JavaScript untuk mengambil (fetch) data dari server atau API. Fetch API memungkinkan kita untuk melakukan permintaan (request) jaringan yang asinkron (non-blocking), artinya kode kita tidak akan dihentikan ketika sedang menunggu respons dari server. Fetch API dapat digunakan di aplikasi web modern dan digunakan untuk melakukan permintaan data dari server, seperti mengambil data JSON atau file gambar.

Dalam implementasinya, Fetch API menggunakan objek Request dan Response untuk memformat permintaan dan respons yang dikirim dan diterima dari server. Selain itu, Fetch API juga mendukung penggunaan promise dan async/await untuk mengatasi kode asinkron.
